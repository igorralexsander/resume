2019

Fullstack Developer - Grupo Hobi mineração de areia e concreto

Recebi uma proposta para retornar a empresa na qual iniciei minha carreira. Nesta nova fase tive como proposta inicial desenvolver um sistema de logística e monitoramento de frota em tempo real. Foi um projeto muito desafiador também, pois a empresa conta com uma frota de aproximadamente 300 veículos. Tive vários desafios em fazer tudo isso funcionar em tempo real. Este sistema contava com um aplicativo Android de logística instalado no celular de cada motorista, onde este trackeava a localização para o Firestore e um backend Java ouvia estas atualizações e via WebSockets comunicava as aplicações front-end mostrando cada veículo no mapa. Depois de cada "viagem" dos veículos, a rota era processada calculando distância, tempo de entrega, tempo de permanência em cada cliente, entre outras métricas que a empresa julgava necessária. Para fazer todos estes cálculos foi utilizado praticamente todas as APIs do Google Maps, DistanceMatrixAPI(estimativa de tempo e distância de chegada), RoadsAPI(Para o ajuste do trajeto encaixando-o na rodovia), DirectionsAPI(para definir a melhor rota), entre outras. Este projeto foi desenvolvido 100% por mim, aplicativo android, aplicação back end em Java(Spring Boot), aplicação front-end em Angular, além da infraestrutura em docker e containers.

Android Kotlin Coroutines Google Maps Retrofit Architeture Components Room Koin Spring Boot Docker Nginx DistanceMatrix RoadsAPI Angular 9 Linux Web Socket Firebase Rest Google Play CSS JavaScript
